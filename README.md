# NeuroVid Studio — Лендинг по оживлению фото

Современный одностраничный сайт (Landing Page) для услуг нейро-реставрации фото и создания вирального контента. Сайт построен с учетом глубокого анализа двух сегментов ЦА: «Хранительницы памяти» (B2C) и «Креаторы-Охотники» (B2B).

## 🛠 Технологический стек

- **Framework:** React 18 + TypeScript
- **Сборка:** Vite
- **Стилизация:** Tailwind CSS
- **Анимации:** Framer Motion
- **Слайдер:** Swiper.js
- **Формы:** React Hook Form + Zod (валидация)
- **Иконки:** Lucide React
- **Уведомления:** Sonner (тосты)

## 🚀 Инструкция по запуску

1.  **Клонируйте репозиторий или скачайте файлы**
    ```bash
    cd neurovid-landing

2. Установите зависимости

bash
npm install

3. Запустите в режиме разработки

bash
npm run dev

Сайт будет доступен по адресу http://localhost:5173.

4. Сборка для продакшена

bash
npm run build
npm run preview

🎯 Особенности реализации

Адаптивность: Mobile-first, проверено на устройствах от 320px до 4K.

Перформанс: Lazy loading изображений и видео через loading="lazy" и Intersection Observer.

Безопасность: Все поля формы проходят санитизацию через Zod перед отправкой.

Чистота кода: Строгое соблюдение принципов DRY и SOLID. Тексты вынесены в config.ts.

Тема: Поддержка светлой и темной темы с сохранением выбора в LocalStorage.
